# avance_proyecto

DOCUMENTACION AVANCE PROYECTO DESARROLLO DE APLICACIONES CON IA


Este es el codigo fuente del agente, estamos trabajando con ollama y el modelo llama3. En el codigo lo primero que hacemos es importar las librerias necesarias, luego le pasamos el documento pdf con el que va a trabajar y el cual es la fuente del conocimiento y de las respuestas, luego divide este documento en fragmentos y vectores para poder analizarlo y de ultimo recibe nuestra pregunta y la procesa para darnos una respuesta.



Esta es la respuesta dada por el modelo, es importante resaltar lo demorado que es en cada interacción por estar instalado localmente en el pc
